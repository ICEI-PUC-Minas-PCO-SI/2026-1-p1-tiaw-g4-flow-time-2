## Justificativa das escolhas
As decisões de design foram baseadas em:
- **Usabilidade**: interface intuitiva e fácil de navegar
- **Público-alvo**: adaptada às necessidades dos usuários definidos no contexto do projeto
- **Acessibilidade**: contraste adequado, fontes legíveis e navegação clara

## Fluxo de navegação
O usuário percorre o sistema da seguinte forma:
1. Tela inicial
2. Login/Registro
3. Acesso às funcionalidades principais
4. Visualização de resultados/relatórios
